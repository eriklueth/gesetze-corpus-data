# § 304 Kostenvorausbelastung

(1) Wurde die Abnahme von Anteilen oder Aktien für einen mehrjährigen Zeitraum vereinbart, so darf von jeder der für das erste Jahr vereinbarten Zahlungen höchstens ein Drittel für die Deckung von Kosten verwendet werden, die restlichen Kosten müssen auf alle späteren Zahlungen gleichmäßig verteilt werden.
