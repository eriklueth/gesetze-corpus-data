# § 461 Mehrere Wiederkaufsberechtigte

(1) Steht das Wiederkaufsrecht mehreren gemeinschaftlich zu, so kann es nur im Ganzen ausgeübt werden. Ist es für einen der Berechtigten erloschen oder übt einer von ihnen sein Recht nicht aus, so sind die übrigen berechtigt, das Wiederkaufsrecht im Ganzen auszuüben.
