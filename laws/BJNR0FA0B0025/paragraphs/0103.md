# § 103 Beschränkung der Beratung über Große Anfragen

(1) Gehen Große Anfragen so zahlreich ein, dass sie die ordnungsgemäße Erledigung der Geschäfte gefährden, so kann der Bundestag zeitweilig die Beratungen darüber auf einen bestimmten wöchentlichen Sitzungstag beschränken. Auch in diesem Fall kann der Bundestag die Beratung über einzelne Große Anfragen an einem anderen Sitzungstag beschließen.
