# § 8 Namenslöschung wegen Nichtausübung freiberuflicher Tätigkeit

(1) Wird der Name einer Partnerschaft gelöscht, weil unter diesem keine freiberufliche Tätigkeit ausgeübt wird, so kann auf Antrag der Gesellschafter in einer Bekanntmachung der Grund der Löschung erwähnt werden.
