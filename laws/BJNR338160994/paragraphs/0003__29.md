# § 3.30 Notzeichen

(1) [1. | Ein in Not befindliches Fahrzeug, das Hilfe durch Sichtzeichen herbeirufen will, kann zeigen:; | - | bei Nacht: | ; | | ein Licht, das im Kreis geschwenkt wird; | ; | - | bei Tag: | ; | | eine rote Flagge, die im Kreis geschwenkt wird, oder einen sonstigen geeigneten Gegenstand, der im Kreis geschwenkt wird. | ; 2. | Diese Zeichen ersetzen oder ergänzen die Schallzeichen nach § 4.04.]
