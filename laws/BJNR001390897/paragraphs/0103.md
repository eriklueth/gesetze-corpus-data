# § 103

(1) Einigen sich die Beteiligten nicht, so macht das Grundbuchamt ihnen einen Vorschlag für eine neue Rangordnung. Es kann hierbei eine Änderung der bestehenden Rangverhältnisse, soweit sie zur Herbeiführung einer klaren Rangordnung erforderlich ist, vorschlagen.
