# § 40 Beschäftigung im Anschluss an das Vertragsverhältnis

(1) Wird die studierende Person im Anschluss an das erfolgreich absolvierte Studium als Hebamme beschäftigt, ohne dass hierüber ausdrücklich etwas vereinbart worden ist, so gilt ein Arbeitsverhältnis auf unbestimmte Zeit als begründet.
