# § 40 Beschäftigung im Anschluss an das Ausbildungsverhältnis

(1) Wird die auszubildende Person im Anschluss an das Ausbildungsverhältnis beschäftigt, ohne dass hierüber ausdrücklich etwas vereinbart worden ist, so gilt ein Arbeitsverhältnis auf unbestimmte Zeit als begründet.
