# § 399 Verzicht auf Zeugen

(1) Die Partei kann auf einen Zeugen, den sie vorgeschlagen hat, verzichten; der Gegner kann aber verlangen, dass der erschienene Zeuge vernommen und, wenn die Vernehmung bereits begonnen hat, dass sie fortgesetzt werde.
