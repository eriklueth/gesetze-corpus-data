# § 175 Beteiligte Rechtsträger

(1) Eine Vollübertragung ist oder Teilübertragungen sind jeweils nur möglich 1.von einer Kapitalgesellschaft auf den Bund, ein Land, eine Gebietskörperschaft oder einen Zusammenschluß von Gebietskörperschaften; 2.a)von einer Versicherungs-Aktiengesellschaft auf Versicherungsvereine auf Gegenseitigkeit oder auf öffentlich-rechtliche Versicherungsunternehmen; b)von einem Versicherungsverein auf Gegenseitigkeit auf Versicherungs-Aktiengesellschaften oder auf öffentlich-rechtliche Versicherungsunternehmen; c)von einem öffentlich-rechtlichen Versicherungsunternehmen auf Versicherungs-Aktiengesellschaften oder auf Versicherungsvereine auf Gegenseitigkeit.
